##

Rock problem

1. Each rock has name with a-z aplhabets.
2. Count the number of aplhabets which is common to all the rocks.


Input:

3
aabbccdd
abde
acbd

Output:
2

Explanation:
a,b are only alphabet common to all the 3 inputs



Process used:
1. Double pointer
2. Malloc to double to pointer.
3. passing and reading string of double pointer.
4. Creating linklist for unique aplhabets in the string.
5. Count the number for output.
